Auflösung

1 → iFrame. Standardelement für externe oder eigene Widgets.

2 → H5P. Liefert fertige Vorlagen; Konto auf h5p.org oder Selbsthosting (Moodle, WordPress).

3 → Custom GPT. Tool für domänenspezifische Tutoren — kein Programmieraufwand nötig.

4 → AcademicCloud. DSGVO-konformer als ChatGPT, Login per Hochschul-Account.

5 → X-Frame-Options. Wenn ein iFrame leer bleibt, ist meist DENY der Grund. Dann nur Link statt Embed.

6 → System-Prompt. Steuert Antwortton, Sprache und Themengrenzen — Lay-Erklärung zuerst, kurze Code-Schnipsel etc.
