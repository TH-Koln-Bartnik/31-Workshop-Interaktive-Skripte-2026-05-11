Auflösung

1 → ::: incremental. Einfachster Weg, eine Bullet-Liste sequenziell aufzubauen.

2 → ::: {.fragment}. Mehrere hintereinander = mehrstufige Argumentationskette.

3 → ## Überschrift. Standardregel: ein H2 pro Folie, eine Idee pro Folie.

4 → # Überschrift. Ein einzelnes H1 markiert Block-Trenner; nutzt das Reveal-Theme automatisch.

5 → ::: notes. Im Plenarmodus unsichtbar, in der Notes-Ansicht parallel zu den Folien — perfekt für Stichworte.

6 → :::: {.columns}. Mit zwei .column-Blöcken darin baut das Spalten-Layout auf.
